**To Add Image Gird in any Shopify themes follow the below steps:**

- Atfirst Goto your Online Theme > open the Edit Code then create the new section file from Section Tabs and named as logo-slider.liquid.
- Copy the code of logo-slider.liquid from the ShopifyResourcePool repo > Logo-Slider folder and paste the code to the created file.
- Ready to use the Logo Slider Section.
- To use the Logo Slider Section
  - Go to theme Customization > Add section > select Logo Slider section.
  - Done.
